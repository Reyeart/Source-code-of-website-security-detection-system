# 云天安全 - 网站安全检测系统

## 系统简介

云天安全提供专业的网站安全检测服务，通过对域名、网站、备案登记信息进行审核，验证网站真实身份。

## 主要功能

### 1. 网站安全检测

- 域名真实性验证
- 网站备案信息核查
- 真实身份认证

### 2. 证书管理

- 数字证书申请
- 证书查询与验证
- 证书状态管理

### 3. 后台管理

- 申请审批流程
- 证书管理
- 系统设置

## 使用流程

1. **提交申请** - 填写网站信息
2. **管理员登录** - 进入系统
3. **等待审核** - 管理员审核中
4. **获取证书** - 审核通过后获得数字证书

## 技术架构

- **前端**: HTML5 + CSS3 + JavaScript
- **后端**: PHP
- **数据库**: MySQL
